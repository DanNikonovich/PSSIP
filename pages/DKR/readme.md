____ ИНСТРУКЦИЯ ПО УСТАНОВКЕ ____

1. Скачать файл DKR
   - Изменить название на  E-ride.local
   - Добавить в корневую папку веб-сервера
   - Для OpenServer: domains/название_проекта/

2. СОЗДАНИЕ БАЗЫ ДАННЫХ
   - Запустить phpMyAdmin
   - Создать новую БД с именем 'e_ride'
   - Импортировать файл e_ride.sql (предварительно скачать из репозитория)

3. НАСТРОЙКА ПОДКЛЮЧЕНИЯ
   - Открыть файлы php
   - При необходимости изменить параметры:
     $host = 'MySQL-8.4:3306';
     $dbname = 'e_ride';
     $user = 'root';
     $pass = '';

4. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
   - Главная страница: http://localhost/название_проекта/
   - Каталог схем: http://localhost/название_проекта/patterns.php
   - Админ-панель: http://localhost/название_проекта/admin/
